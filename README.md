# Claude Agentic Handbook

A comprehensive handbook for learning and effectively using **Claude Code**, **agentic software engineering**, **Model Context Protocol (MCP)**, and AI-powered development workflows.

📖 **Read the handbook online:**

**https://oshinraj.github.io/claude-agentic-handbook/agentic-claude-code-handbook.html**

## Overview

This handbook serves as a practical reference for developers who want to:

* Learn Claude Code workflows
* Understand agentic software engineering concepts
* Build and use specialized AI agents
* Work with MCP (Model Context Protocol)
* Apply best practices for AI-assisted software development
* Improve productivity with autonomous coding workflows

## Access
